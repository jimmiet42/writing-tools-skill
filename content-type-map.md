# Content type to tool mapping

Look up the content type the user selected and load only the listed tool numbers. Refer to `tools-index.md` for each tool's principle and when to reach for it.

## 1. Email (internal or external)

Intent: short, direct, human. Get to the point fast.

- Tool #1. Branch to the Right
- Tool #2. Use Strong Verbs
- Tool #3. Beware of Adverbs
- Tool #11. Back Off or Show Off
- Tool #34. Cut Big, Then Small
- Tool #38. Polish Your Jewels
- Tool #39. The Voice of Verbs

## 2. Memo, report, or internal doc

Intent: structured, skimmable, clear. Reader scans first, reads second.

- Tool #9. Prefer Simple to Technical
- Tool #12. Control the Pace
- Tool #24. Name the Big Parts
- Tool #30. Write Endings to Lock the Box
- Tool #34. Cut Big, Then Small
- Tool #36. Write a Mission Statement
- Tool #42. Paragraphs (short or long by purpose)

## 3. Slide notes or presentation copy

Intent: punchy, spoken-friendly, rhythm that reads out loud well.

- Tool #2. Use Strong Verbs
- Tool #4. Period as Stop Sign
- Tool #12. Control the Pace
- Tool #25. Purposeful Repetition
- Tool #38. Polish Your Jewels

## 4. Short-form copy (headlines, subject lines, CTAs, button text)

Intent: every word earns its place. Cut anything that does not carry weight.

- Tool #2. Use Strong Verbs
- Tool #4. Period as Stop Sign
- Tool #8. Seek Original Images
- Tool #14. Interesting Names
- Tool #16. Odd and Interesting Things Next to Each Other
- Tool #25. Purposeful Repetition
- Tool #38. Polish Your Jewels

## 5. Product or landing page copy

Intent: clarity plus pull. Reader skims the top, then commits to the body.

- Tool #1. Branch to the Right
- Tool #2. Use Strong Verbs
- Tool #9. Prefer Simple to Technical
- Tool #13. Show and Tell (ladder of abstraction)
- Tool #25. Purposeful Repetition
- Tool #38. Polish Your Jewels

## 6. Explainer, guide, article, or blog post

Intent: hold attention over a longer read. Anchor big ideas in concrete detail.

- Tool #9. Prefer Simple to Technical
- Tool #12. Control the Pace
- Tool #13. Show and Tell
- Tool #18. Internal Cliffhangers
- Tool #23. Place Gold Coins Along the Path
- Tool #24. Name the Big Parts
- Tool #30. Write Endings to Lock the Box
- Tool #36. Write a Mission Statement
- Tool #42. Paragraphs (short or long by purpose)
